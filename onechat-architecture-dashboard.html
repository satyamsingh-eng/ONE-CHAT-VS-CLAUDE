Can Be Improved

🔧 Refinements
  • High

    AI-Powered Memory Relevance Selection

    Side-query to Gemini Flash: "Pick 5 most relevant memories for this query." Scans all memory files, selects contextually relevant ones. Reduces noise in system prompt. Currently retrieves memories but no AI-driven relevance filtering — may inject irrelevant memories into context.

    backend/integrations/pinecone_client.py
  • Medium

    Context Compression Strategy

    No explicit context size limits for memory injection. Risk of context window overflow. Add memory content limits in orchestrator. Summarize long memories before injection. Cap total memory context to 10% of model's context window.

  • Low

    Memory Type Taxonomy

    Currently single memory type. Claude Code uses 4 types: user (preferences, role, goals), feedback (corrections, confirmations), project (ongoing work context), reference (external systems, docs). Tag memories by type for better retrieval.
